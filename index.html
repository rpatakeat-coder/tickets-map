Estrutura transversal
Barramento de propriedades-chave
Cada propriedade do HubSpot dispara um ou mais workflows do n8n. Esta é a "API" implícita entre os sistemas.
gatilho_disparos_setup
→ Setup Dispairs (roteador com 7 ramos: aviso novo cliente, validação cardápio, ajuste cardápio, sem cardápio, agendar config, lembrete reunião, conclusão).
disparar_acessos
→ Enviar acessos grupo Whatsapp (quando = Sim).
mensagem_disparada
→ Send Message SquadLead New Menu (quando = "Disparo Novo Cardápio Recebido").
novo cardápio
→ Setup ↔ Docupipe (quando = SIM). Menu IA Mountage ainda manual.
disparo_treinamento
→ Send Link Setup Ativacao (polling: busca onde está vazio).

Propriedades-chave que conectam HubSpot ↔ n8n
Ticket properties usadas como gatilhos ou dados de entrada:
gatilho_disparos_setup — roteador principal do Setup Dispairs (7 valores possíveis)
Disparar Acessos — dispara Enviar acessos grupo Whatsapp
gatilho novo cardápio — dispara integração Docupipe
disparo_treinamento — controla envio de links de treinamento
cardapio_anexado — IDs dos arquivos para Docupipe e Menu IA
username_no_dash — usado em mensagens de cardápio pronto e acessos
email_no_dash, senha_no_dash — credenciais enviadas ao cliente
tipo_de_setup — filtro para tickets de Cardápio

Deal properties:
waliid — ID do grupo WhatsApp do cliente (usado em quase todos os n8n)
celular — fallback para envio privado
vendedor — cruzado com Supabase para nome do vendedor
mensagem_disparada — valor avaliado pelo Send Message SquadLead New Menu
cardapio_da_loja — URL do arquivo do cardápio
Pontos críticos transversais (consolidado de todos os workflows)
Segurança: senha do cliente em texto claro no WhatsApp e na descrição do grupo (Enviar acessos), credenciais hardcoded no Menu IA e Docupipe, planilhas com permissão pública de edição.

Dívida técnica: 17 nodes Evolution API órfãos no Setup Dispairs, 9 no Enviar acessos, 6 no Menu IA. Vários números e IDs de grupo hardcoded sem fallback.

Robustez: Setup Dispairs e Menu IA não têm Error Trigger. Bug confirmado: Onboarding Group = Mkt Group. cardapio_da_loja usado sem ser solicitado no Send Message SquadLead. Verificação de participação no grupo por últimos 8 dígitos pode dar falsos positivos.

Operação: Menu IA ainda em trigger manual. Send Link Setup Ativacao é o único self-triggered. Os demais n8n dependem de Execute Workflow chamado por outro fluxo (geralmente um workflow do HubSpot que aciona via webhook).
Quem dispara quem? (visão de causalidade)
HubSpot dispara n8n via:
• Workflow HubSpot edita uma property do ticket/deal → webhook → Execute Workflow Trigger no n8n
• Propriedade modificada: gatilho_disparos_setup, Disparar Acessos, gatilho novo cardápio, mensagem_disparada

n8n self-triggered:
• Send Link Setup Ativacao — Schedule Trigger (a cada 2 min)

n8n chama n8n (sub-workflows):
• Instance Selector — escolhe token UAZAPI correto
• Horas Economizadas — registra ganho operacional
• Métrica Erros Painel RPA — registra falhas

Docupipe chama n8n via webhook:
• /documento_processado — quando OCR termina
• /documento_standarizado — quando padronização termina
